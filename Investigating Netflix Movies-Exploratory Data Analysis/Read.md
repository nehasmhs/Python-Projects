This project focuses on **exploratory data analysis (EDA)** of Netflix movie data to uncover insights about movies released in the **1990s decade**.  
The analysis is designed from the perspective of a **production company** that wants to understand historical trends in movie duration and genre-specific patterns.

Using the dataset `netflix_data.csv`, we analyze:
- Common movie durations in the 1990s  
- Trends in short movies (less than 90 minutes)  
- Genre-based insights, specifically **Action movies**
